# Changelog

All notable changes to PopReklam will be documented in this file.

## [1.0.0] - 2026-02-08

### Initial Release

#### ✨ Features
- Complete ad network platform with Publisher and Advertiser dashboards
- User authentication with JWT and role-based access
- Publisher site management with real-time statistics
- Advertiser campaign management with targeting options
- Payment processing for both publishers and advertisers
- RESTful API with comprehensive endpoints
- Modern, responsive UI with glassmorphism design
- MySQL database with Prisma ORM
- Enhanced seed script for demo data generation

#### 📊 Publisher Features
- Dashboard with analytics
- Multi-site management
- Ad code generation
- Payment withdrawal requests
- Performance statistics
- Profile and settings management

#### 📢 Advertiser Features
- Campaign creation and management
- Budget control (daily and total)
- Advanced targeting (geo, device, OS, browser)
- Multiple ad formats (Popunder, In-Page Push, Native)
- Billing and fund management
- Real-time campaign statistics

#### 🛠️ Technical
- Next.js 14 with App Router
- React 18 with Server Components
- Express.js backend
- Prisma ORM for database
- TailwindCSS for styling
- Lucide React for icons

#### 📝 Documentation
- Complete installation guide
- User guide for publishers and advertisers
- API documentation
- License agreements
