# MrPop.io - Premium Ad Network Platform

![Version](https://img.shields.io/badge/version-1.0.0-blue) ![License](https://img.shields.io/badge/license-Commercial-green) ![Next.js](https://img.shields.io/badge/Next.js-14-black) ![Node.js](https://img.shields.io/badge/Node.js-18+-green)

**Professional advertising network platform with separate Publisher and Advertiser dashboards for monetizing website traffic and running ad campaigns.**

---

## ✨ Key Features

### For Publishers
- 📊 **Real-time Analytics** - Track impressions, clicks, revenue, and CPM
- 🌐 **Multi-Site Management** - Manage unlimited websites from one dashboard
- 💰 **Flexible Payments** - PayPal, Wire Transfer, Bitcoin withdrawals
- 📝 **Easy Integration** - Simple ad code implementation
- 📈 **Performance Reports** - Detailed statistics and insights

### For Advertisers
- 🎯 **Advanced Targeting** - Geographic, device, browser, OS targeting
- 📢 **Multiple Ad Formats** - Popunder, In-Page Push, Native ads
- 💳 **Budget Control** - Set daily and total budgets
- 📊 **Campaign Analytics** - Real-time performance tracking
- ⚡ **Quick Approval** - Fast campaign review process

### Platform Features
- 🔐 JWT Authentication with role-based access
- 🎨 Modern, responsive UI with glassmorphism design
- 🚀 High-performance backend with Prisma ORM
- 📱 Mobile-friendly dashboards
- 🔄 Real-time data updates
- 🎯 RESTful API architecture

---

## 🛠️ Tech Stack

| Layer | Technology |
|-------|------------|
| **Frontend** | Next.js 14, React 18, TailwindCSS |
| **Backend** | Node.js, Express.js 4.x |
| **Database** | MySQL 8.0+ with Prisma ORM |
| **Authentication** | JWT (JSON Web Tokens) |
| **Icons** | Lucide React |
| **Styling** | TailwindCSS with custom glassmorphism |

---

## 📋 Requirements

- **Node.js** 18.0 or higher
- **MySQL** 8.0 or higher  
- **npm** or **yarn** package manager
- **Modern web browser** (Chrome, Firefox, Safari, Edge)
